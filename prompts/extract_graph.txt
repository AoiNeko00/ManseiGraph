
-Goal-
Given a text document that is potentially relevant to this activity and a list of entity types, identify all entities of those types from the text and all relationships among the identified entities.

-Domain-Specific Rules-
This data covers Korean independence movement history (한국 독립운동사). Apply the following rules:

1. **Homonym Disambiguation (동명이인 분리):**
   - 반드시 성+이름 전체(FULL NAME)를 사용하라. "김" "이" 등 성만 쓰지 말 것.
   - 동일 성씨·유사 이름 인물 구분에 주의:
     - 김구(백범) vs 김규식(우사) — 둘 다 임시정부 핵심이나 역할이 다름
     - 이승만(외교론) vs 이동휘(사회주의) vs 이회영(아나키즘) — "이" 씨 독립운동가 다수
     - 김좌진(청산리) vs 김원봉(의열단) — 무장투쟁 계열이나 노선 상이
   - 동일 텍스트에 같은 성의 인물이 2명 이상 등장하면, 첫 언급 시 "(호: ○○)" 또는 소속 조직을 entity_description에 반드시 병기하라.
   - 모호한 대명사("그", "그녀", "장군")는 반드시 실명으로 치환하여 추출하라.

2. **Symbolic Figure Relationships (상징적 인물·사건의 관계 강조):**
   - 독립운동사의 핵심 인물(김구, 안창호, 안중근, 유관순, 윤봉길 등)은 단순 소속 관계를 넘어 **사상적 영향, 계승 관계, 대립 구도**까지 포착하라.
   - 예: 김구가 의열단 활동에 고무되어 한인애국단을 조직한 관계, 신채호가 이승만의 외교론을 비판하고 무장투쟁론을 제시한 관계.
   - 상징적 사건(3·1 운동, 청산리 전투, 하얼빈 의거 등)이 후속 조직/인물에 미친 영향도 relationship으로 추출하라.
   - relationship_strength 기준: 직접 행위(의거 수행, 조직 창설)=9~10, 사상적 영향/계승=7~8, 간접 관련=5~6.

3. **Alternative Names (이명/호):** Many figures have pen names (호), courtesy names, or aliases (e.g., 김구 = 백범, 안창호 = 도산, 신채호 = 단재, 김원봉 = 약산, 김좌진 = 백야). When the text uses an alternative name, map it to the canonical name and note the alias in the description. 절대 호(號)를 별도 엔티티로 분리하지 말 것.

4. **Organization Evolution (조직 변천):** Independence movement organizations frequently merged, split, or evolved. 주요 계보:
   - 의열단 → 조선민족혁명당 → 조선의용대 → (일부) 한국광복군 합류
   - 신민회 → 서간도·북간도 독립군 기지 → 북로군정서·서로군정서
   - 대한광복단 → 대한정의단 → 군정부 → 북로군정서
   이러한 전환을 "EVOLVED_INTO", "MERGED_WITH", "SPLIT_FROM" 등의 의미를 relationship_description에 명시하라.

5. **Date Precision:** Extract specific dates when available (e.g., 1909-10-26 for 안중근 의거). Include dates in entity and relationship descriptions.

6. **Colonial Context:** Entities related to Japanese colonial rule (조선총독부, 동양척식주식회사, etc.) should be typed appropriately and their oppressive role noted in descriptions.

7. **Cross-Document Entity Consistency (문서 간 엔티티 일관성):**
   - 동일 인물·조직은 항상 동일한 entity_name을 사용하라. 예: "대한민국 임시정부" (O) vs "임시정부" (X) vs "임정" (X).
   - 정식 명칭 기준: 대한민국 임시정부, 한국광복군, 조선총독부, 동양척식주식회사.

8. **Ideological Lines (사상 노선 구분):**
   - 외교독립론(이승만), 무장투쟁론(김원봉, 김좌진), 실력양성론(안창호), 사회주의(이동휘) 등 노선 차이를 relationship_description에 반영하라.
   - 노선 간 협력·갈등·전환도 관계로 추출하라.

-Steps-
1. Identify all entities. For each identified entity, extract the following information:
- entity_name: Name of the entity, capitalized
- entity_type: One of the following types: [{entity_types}]
- entity_description: Comprehensive description of the entity's attributes and activities
Format each entity as ("entity"<|><entity_name><|><entity_type><|><entity_description>)

2. From the entities identified in step 1, identify all pairs of (source_entity, target_entity) that are *clearly related* to each other.
For each pair of related entities, extract the following information:
- source_entity: name of the source entity, as identified in step 1
- target_entity: name of the target entity, as identified in step 1
- relationship_description: explanation as to why you think the source entity and the target entity are related to each other
- relationship_strength: a numeric score indicating strength of the relationship between the source entity and target entity
 Format each relationship as ("relationship"<|><source_entity><|><target_entity><|><relationship_description><|><relationship_strength>)

3. Return output in Korean as a single list of all the entities and relationships identified in steps 1 and 2. Use **##** as the list delimiter.

4. When finished, output <|COMPLETE|>
